Tkinter

Tkinter es una biblioteca gráfica y es usada para crear interfaces gráficas para nuestros proyectos desarrollados con Python. A continuación se presentara todo lo básico de esta herramienta desde cómo crear una ventana, botón, etiquetas, cajas de texto, ventanas emergentes, radiobuttom, checkbuttom, menús y más.

Ventana Grafica

from tkinter import *

root = Tk()
root.geometry("300x300")
root.title("Hello World")

root.mainloop()

Icono a la ventana

root.iconbitmap("image.ico")

Redimensión de ventana

root.resizable(0,0)

Botón

from tkinter import *

root = Tk()
root.geometry("300x300")
Button(root, text="Click Here").pack()

root.mainloop()

Color de fondo para el botón

Button(root, text="Click", bg="#3174B6").pack()

Color al texto del botón

Button(root, text="Click", fg="#3174B6").pack()

Menú

from tkinter import *

root = Tk()
menubar = Menu(root)
root.config(menu=menubar)

filemenu = Menu(menubar, tearoff=0)
filemenu.add_command(label="New")
filemenu.add_command(label="Open")
filemenu.add_command(label="Save")
filemenu.add_separator()
filemenu.add_command(label="Exit", command=root.quit

editmenu = Menu(menubar)
helpmenu = Menu(menubar)

menubar.add_cascade(label="File", menu=filemenu)
menubar.add_cascade(label="Edit", menu=filemenu)
menubar.add_cascade(label="Help", menu=filemenu)

root.mainloop()

Tipos de ventanas emergentes

Ventana emergente con un mensaje de tipo advertencia
messagebox.showwarning("window title","message in the window")
Ventana emergente con opcion de respuesta de reintentar o cancelar
messagebox.askretrycancel("window title","message in the window")
Ventana emergente con opcion de respuesta de Si o No
messagebox.askyesno("window title","message in the window")
Ventanas emergente con opcion de respuesta aceptar o cancelar
messagebox.askokcancel("window title","message in the window")
Ventanas emergente con opcion de respuesta Si o No
messagebox.askquestion("window title","message in the window")
Ventana emergente con un mensaje de error
messagebox.showerror("window title","message in the window")

Posicionamiento Relativo (Pack)

El posicionamiento de elementos con pack empaqueta el elemento en relación al elemento que lo contiene.

Ejemplo de uso

button = Button(root, text="Click").pack()

Posicionamiento Adsoluto (Place)

El posicionamiento con place nos permite colocarle coordenadas X, Y en que queremos que se situé nuestro elemento

Ejemplo de uso

Label(root, text="Hello World").place(x=30, y=60)

padx, pady

Esta propiedad sirve para establecer márgenes fuera de un elemento

Ejemplo de uso

Button(root, text="Click").pack(padx=80, pady=80)

ipadx, ipady

Estas propiedades sirven para establecer márgenes a lo interno de un elemento.

Ejemplo de uso

Button(root, text="Click").pack(ipadx=80, ipady=80)

Posicionamiento con Side

Esta propiedad se utiliza para establecer el lado en que queremos que se situé un elemento se determina con Top, Bottom, Left, Right.

Ejemplo de uso

Button(root, text="Click").pack(side="left")
